What DARIO Does

Everything an AI team needs to work autonomously — in one install.

⚡

Durable Execution Engine

Tasks go through a 12-step pipeline: guardrails validate, context is injected, rubric is generated, trace starts, Claude API executes with optimal model routing (Haiku/Sonnet/Opus), LLM-as-Judge scores, replanner handles failures, state advances atomically in SQLite.

Core
🧠

Self-Evolving

Learning journals capture patterns. Synaptic weights evolve. High-performing skill pairs get dispatch priority. The system literally rewrites its own routing logic.

AI-Native
🔄

Auto-Recovery

Task fails? Replanner retries with same worker, tries a sibling, falls back to alternative skill, or escalates. Pipeline never stops.

Resilient
🛡️

7 Pre-Execution Guardrails

Before ANY task executes: required fields verified, skill exists in hierarchy, budget sufficient, dependencies met, assignee valid, state allows execution, no double-execution. Blocks if FAIL.

Safety
📊

Adaptive Quality Scoring

16 skill-specific rubric profiles. Brand tasks scored on creativity (30%), financials on accuracy (35%). LLM-as-Judge via Haiku at $0.001/score. Not fixed 5 dimensions — contextual.

Intelligence
🔮

Predictive Dispatch

Before executing: "This task will score ~84/100, cost ~$0.02, 0% revision risk. Recommend Haiku model." Uses historical data for estimates.

Intelligence
🗄️

SQLite ACID

No more YAML race conditions. WAL mode, CAS checkout, atomic transactions. Crash recovery via write-ahead log.

Persistence
🌐

45 REST Endpoints

FastAPI runtime on :8422. Tasks, dispatch, state, audit, budget, scores, chains, templates, plugins, compliance, metrics, NL queries.

API
🔗

DAG Chains

Skill chains with parallel waves, per-step checkpoints, structured artifact validation. Resume from any point after crash.
