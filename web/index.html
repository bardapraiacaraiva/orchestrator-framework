Features

Everything you need to orchestrate AI

A production-grade control plane that turns a single Claude Code instance into a coordinated, self-managing AI workforce.

Atomic Task Management

Persistent YAML taskboard with atomic checkout, ownership tracking, and status lifecycle. No task is ever lost or double-assigned.

Quality Scoring

5-dimension auto-evaluation (specificity, actionability, completeness, accuracy, tone) scores every output 0-100. Below 60 triggers revision.

Budget Enforcement

Real-time token budget tracking. Warning at 80%, hard stop at 95%. Never blow through your API spend again.

SLA Timeouts

Define time limits per task type. Auto-escalation on breach ensures nothing stalls silently in the queue.

Auto-Playbook

Domain detection identifies your industry and assembles the optimal skill chain automatically. Zero manual configuration needed.

Error Recovery

Automatic retry with exponential backoff. Failed tasks go to a dead-letter queue for manual review. Nothing disappears.

How it works

Up and running in 3 steps

From zero to autonomous AI orchestration in under 5 minutes.

Install

One command sets up the framework, creates your taskboard, and initializes config files.

npx orchestrator-ai-framework init

Configure

Add your skills and drop in a company.yaml describing your team, domains, and preferences.

skills/ + company.yaml

Run

Launch autopilot mode. The orchestrator scans, dispatches, executes, and scores autonomously.

/orch-autopilot
